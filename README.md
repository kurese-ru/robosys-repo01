# モンスタークリエイト
![test](https://github.com/kurese-ru/robosys-repo01/actions/workflows/test.yml/badge.svg)

標準入力から読み込んだ文字をテンプレートに挿入し、標準出力として文章を生成する。
- ダウンロード方法
1.画面右上にある Code をクリック。
2.HTTPSを選択し、文字列をコピー。
3.Linux上で</>git clone 文字列</>を入力。
4.</>cd リポジトリ名</>で移動。
## 必要なソフトウェア
- Python
  - テスト済みバージョン: 3.7〜3.10
## テスト環境
- Ubuntu 20.04 LTS
